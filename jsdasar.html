Materi Javascript

Javascript merupakan bahasa pemrograman yang dikembangkan unutk membuah sebuah website agar lebih interaktif dan dinamis. Javascript sendiri merupakan bahasa pemrograman interpreter sehingga tidak memerlukan proses compiling untuk menjalankan. Bahasa javascript memiliki fitur seperti object-oriented, client-side, pemrograman level tinggi dan loosy typed.


Untuk menjalankan javascript terdapat 2 cara yaitu, menjalankan langsung melalui console browser dan menjalankan lewat file yang di muat pada sebuah halaman web.


Secara umum struktur kode javascript atau yang bis akita sebut dengan script terditri tiga bagian yaitu :

  1. Statement
    Sebuah statement bisa kita sebut sebagai kalimat tapi dalam kode. Statement adalah semua syntax dan perintah-perintah yang menjalankan aksi.
  2. Semi colon (;)
    Pada javascript titik koma tidak menjadi sebuah kewajiban jika antara statement satu dengan lainnya dipishakan dengan baris baru. Namun jika setiap statement ditulis dalam satu baris maka wajib diakhiri dengan titik koma.
  3. Komentar
    Semua statement yang dibungkus dengan komentar tidak akan dijalankan. Penulisan komentar dalam javascript ada dua, yakni :
    • Komentar satu baris. Sebuah komnetar satu baris ditandai dengan menambahkan “//” di awal statement.
    • Komentar multi baris. Komentar multi baris dibungkus dengan tanda “/*…..*/”.


Penjelasan di atas merupakan bagian pendahuluan sebelum kita mempelajari lebih dalam mengenai dasar JavaScript. Adapun materi yang akan kita pelajari pada modul ini sebagai berikut :

  • Tipe Data

Tipe Data

Data type atau tipe data dalam bahasa pemrograman adalah sebuah klasifikasi yang menentukan suatu tipe dari nilai. Tipe data ini juga menentukan operasi matematika, relasional maupun operasi logika apa saja yang dapat digunakan tanpa menyebabkan kesalahan kode. Dalam Javascript terdapat 8 jenis tipe data :


Number

Tipe data number dalam Javascript digunakan untuk mempresentasikan integer (bilangan bulat) dan float (bilangan pecahan).
Ada beberapa operasi bilangan yang dapat digunakan seperti, perkalian “*”, pembagian “/”, penjumlahan “+”, penguranagn “-” dan sebaginya.


Selain bilangan regular (integer dan float) terdapat juga nilai khusus yang masuk dalam tipe data number. Nilai khusus tersebut adalah

  • Infinity | -Infinity
    (representasi nilai tak berhingga)
    Diperoleh dengan membagi bilangan apapun dengan angka 0
  • NaN
    (representasi kesalahan perhitungan)

BigInt

Dalam Javascript tipe data number tidak bisa mempresentasikan bilangan bulat lebih dari (2^53-1) atau bernilai 9007199254740991 atau untuk bilangan negatif -(2^53-1).
Menggunakan fungsi bigInt ...


Sebuah nilai yang bertipe BigInt selalu diakhiri dengan huruf n pada akhir sebuah nilai integer.

String

Penulisan tipe data string harus dibungkus dengan tanda petik. Dalam javascript terdapat tiga jenis tanda petik.

  • "Hello World"
  • 'Hello World'
  • `Hello World`
Tidak ada perbedaan antara penggunaan tanda petik ganda dan petik tunggal.
Namun, untuk tanda backtick (``) biasanya untuk menggabungkan variable dan ekspresi serta harus dibungkus dengan ${….}.
Contoh :
const word = "Dunia"
const uji_string = document.getElementById("uji-string");
uji_string.textContent = `Hello, ${word}`;

Boolean

Bollean adalah tipe data berjenis logikal. Tipe data ini hanya memiliki dua nilai, true atau false. True artinya benar dan false artinya salah. Nilai tipe data Boolean dapat juga dihasilkan dari sebuah hasil perbandingan.

Contoh : membandingkan 2 angka




Null

Null merupakan sebuah nilai khusus yang bukan milik tipe data apapun. Pada javascript null bukan referensi untuk sebuah object yang tidak ada atau sebuah null-pointer seperti kebanyakan bahasa pemrograman lain. Null hanya sebuah nilai khusus yang mempresentasikan nilai yang tidak ada isinya, kosong, atau nilai tidak diketahui.
Contoh :

const nilai = null;
const uji_null = document.getElementById("uji-null");
uji_null.textContent = nilai;

Undefined

Undefined merupakan nilai khusus yang mempresentasikan bahwa sebuah variable belum memiliki nilai yang di-assign terhadapnya. Artinya sebuah variabel sudah di deklasrisakn sebelumnya namun belum memilki sebuah nilai yang kemudian kita akses atau lakukan operasi.
Contoh :

let nama;
const uji_undefined = document.getElementById("uji-undefined");
uji_undefined.textContent = "Output : " + nama;

Contoh :

const prodi = Symbol('Teknik Informatika');
document.getElementById("output-uji-symbol").textContent = prodi.toString();
document.getElementById("uji-symbol").textContent = prodi.description.toString();

Object

Tipe data objek adalah tipe data khusus yang dapat menyimpan kumpulan nilai dengan kunci/ key. Sebuah objek dapat didefiniskan dengan menggunakan kurung kurawal “{…}” dengan property yang sifatnya opsional di dalamnya. Sebuah property dalam objek adalah pasangan dari sebuah “key : value”. Dimana “key” berupa string (key bisa juga disebut sebagai property name) dan “value” bisa bernilai apapun. Untuk mengakses sebuah nilai dalam objek kita dapat menggunakan notasi “.” Dan diikuti key/ nama propertinya

Operator Ternary

Operator ternary merupakan operator yang terdiri dari tiga bagian (kiri, tangah dan kanan).

Operator-operator sebelumnya disebut operator binary karena hanya memiliki 2 bagian saja. Sedangkan untuk operator yang hanya memiliki 1 bagian disebut operator unary.

Operator ternary digunakan untuk melakukan percabangan (if-else). Cara penggunaan :

kondisi ? jika benar : jika salah

Contoh :
let a = 1;
let b = 1;
console.log( a == b ? "yes" : "no");


Conditional Branching

If - Else If - Else

If, digunakan untuk mengevaluasi suatu kondisi. Jika kondisi tersebut benar (true), maka blok kode di dalamnya akan dieksekusi. Jika tidak maka akan dilewati.

if (kondisi) {
# Blok kode yang dieksekusi jika kondisi bernilai True
}

Else If, digunakan untuk menambahkan kondisi alternatif setelah if.

if (kondisi1) {
# Blok kode yang dieksekusi jika kondisi1 bernilai True
} else if (kondisi2) {
# Blok kode yang dieksekusi jika kondisi2 bernilai True
}

Else, digunakan sebagai bagian terakhir dari struktur pengkondisian.

if (kondisi1) {
# Blok kode yang dieksekusi jika kondisi1 bernilai True
} else if (kondisi2) {
# Blok kode yang dieksekusi jika kondisi2 bernilai True
} else {
# Blok kode yang dieksekusi jika tidak ada kondisi sebelumnya yang terpenuhi
}


Contoh : diketahui blok kode sebagai berikut

if (kondisi <= 100 && kondisi >= 90) {
Anda mendapat A
} else if (kondisi <= 89 && kondisi >=80) {
Anda mendapat B
} else if (kondisi <=79 && kondisi >=70) {
Anda mendapat C
} else {
Anda mendapat D
}

Switch Statement

Switch dapat digunakan untuk menggantikan if else yang memiliki banyak percabangan sehingga lebih deskriptif unutk membandingkan banyak variansi nilai.


Struktur dari switch statement terdiri dari beberapa case (kasus) yang mewakili kemungkinan nilai variabel yang dapat diuji. Ketika nilai variabel sesuai dengan salah satu case, blok kode yang sesuai dengan case tersebut akan dieksekusi. Switch statement juga biasanya memiliki satu atau lebih case default yang dieksekusi jika tidak ada case yang sesuai dengan nilai variabel.


Penggunaan switch statement hanya cocok untuk kasus yang perlu memeriksa nilai variabel terhadap beberapa nilai diskrit. Jika diperlukan evaluasi kondisi yang lebih kompleks atau dengan syarat-syarat yang rumit, maka penggunaan if-else statement akan lebih sesuai.


Contoh : diketahui blok kode sebagai berikut

switch (kondisi) {
case 10:
case 9:
hasil = "Sangat Baik";
break;
case 8:
case 7:
hasil = "Baik";
break;
case 6:
case 5:
hasil = "Cukup";
break;
default:
hasil = "Kurang";
break;
}


Looping

Loop atau perulangan adalah proses untuk menjalankan perintah secara berulangan.

For Loop

merupakan struktur pengulangan yang digunakan untuk mengeksekusi blok kode secara berulang berdasarkan kondisi yang ditentukan.


For loop cocok digunakan ketika Anda sudah mengetahui berapa kali kode akan diulang.

Contoh :

for (let i = 0; i < 5; i++) {
i
}
Output :

While Loop

digunakan untuk mengeksekusi blok kode secara berulang selama kondisi yang ditentukan bernilai benar (true).


Cocok digunakan ketika tidak tahu berapa kali kode akan diulang, namun akan berhenti saat kondisi tertentu.

Contoh :

let j = 1;
while (j <= 5) {
j++ }
Output :

Do While

hampir sama dengan while loop, namun blok kode akan dieksekusi setidaknya sekali sebelum kondisi diuji.


Cocok digunakan untuk memastikan bahwa blok kode di dalamnya dieksekusi setidaknya sekali, bahkan jika kondisi awalnya tidak memenuhi.

Contoh :

let k = 1; do {
k++;
} while (k = 5);

Statement Tambahan Pada Looping

Break

digunakan untuk mengkontrol sebuah perulangan dengan menghentikan perulangan.

Contoh :

for (let a = 1; a <= 5; a++) {
if (a == 3) {
break
}
a }
Output :

Continue

digunakan untuk meloncati iterasi sekarang dan melanjutkan ke iterasi selanjutnya

Contoh :

for (let a = 1; a <= 5; a++) {
if (a == 3) {
continue
}
a }

Pop Up Box

Javascript secara default memiliki tiga jenis popup box yaitu alert, prompt dan confirm.

Alert

Alert biasa digunakan unutk menampilkan pesan dan untuk mendapatkan perhatian dari pengguna.
Popup box jenis alert akan menampilkan pesan dengan sebuah tombol “OK”.




Prompt

Prompt sering digunakan jika kita ingin pengguna menginputkan sebuah nilai terlebih dahulu sebelum memasuki sebuah halaman.
Ketika prompt box tampil, pengguna harus klik “OK” atau “CANCEL” setelah memasukkan nilai.
Jika pengguna klik “OK” maka prompt akan mengembalikan nilai yang dimasukkan. Jika pengguna klik “CANCEL” maka prompt akan mengembalikan nilai null.


Confirm

Confirm digunakan untuk menampilkan pesan konfirmasi kepada pengguna. Confirm box menampilkan pesan dengan dua buah tombol “OK” dan “CANCEL”. Confirm mengembalikan nilai true jika kita tekan “OK” dan false jika kita tekan “CANCEL”.
